# Luxe Flame Website

This site now loads product data from `products.json` and renders the product catalog dynamically.

## Admin workflow

- Open `admin.html` in your browser.
- Log in using the default admin password: `LuxeFlame123`
- Edit product fields or add/remove products.
- Click `Download products.json` after making updates.
- Replace the `products.json` file in the repository with the downloaded file.
- Commit and push changes to GitHub.

## Deploy

- Host the repository on GitHub Pages.
- Your product data is now separated from the HTML, so adding/removing products is easier.

## Notes

This admin editor is static and does not automatically push changes to GitHub. To publish updates, you must commit the new `products.json` to the repository.
